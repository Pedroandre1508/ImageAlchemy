# Projeto de Processamento Digital de Imagens - PDI 2025

Este é um projeto-base em C++ que desenvolve algoritmos de processamento de imagens no contexto acadêmico.

## ⚙️ Execução

### 🐧 Linux
```bash
cd pdi_code/build
cmake ..
make
./pdi_m2
```

### 🪟 Windows
```powershell
cd pdi_code
mkdir build
cd build
cmake ..
cmake --build . --config Release
.\Release\pdi_m2.exe
```
