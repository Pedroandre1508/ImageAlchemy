# Projeto de Processamento Digital de Imagens - PDI 2025

Este é um projeto-base em C++ com OpenCV para implementar algoritmos de processamento de imagens no contexto acadêmico.

## 🎯 Estrutura do projeto

```
projeto/
├── CMakeLists.txt
├── app/
│ └── run.cpp
├── include/
│ └── (arquivos .hpp)
├── src/
│ └── (arquivos .cpp)
└── data/
└── (imagens para teste)
```

---
